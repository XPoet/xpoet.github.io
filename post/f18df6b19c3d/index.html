准备工作

首先确保在你的 Mac 上已安装 Git。

在终端输入命令git --version，能打印出具体的版本号，表示 Git 正确安装。

开始配置

本文以配置 GitHub 和 GitLab 为案例，生成两个公共/私有 rsa 密钥对：rsa_github 和 rsa_gitlab。

创建 rsa 密钥对

  1. 进入到当前用户目录

  2. 创建 .ssh 目录

    如果 .ssh 目录已经存在，则忽略这一步。

  3. 进入到 .ssh 目录

  4. 生成公共/私有 rsa 密钥对

    此时会看到终端提示输入要保存密钥的文件名，为了做区分，我们给文件名加个后缀，本例第一个 rsa 密钥对：rsa_github。

    接着会看到终端提示输入密码，敲两次 Enter 回车键 跳过输入密码。

    最终会在 .ssh 目录 里生成 rsa_github 和 rsa_github.pub 密钥对。

  5. 参照第 4 步，创建本例的第二个 rsa 密钥对，rsa_gitlab。

使用 SSH Key

按照上面步骤，成功创建并且配置 rsa 密钥对，接下来就可以在对应的 Git 平台网站使用 SSH Key，完成和自己 Git 账号的关联。

GitHub

GitHub 添加 SSH Key 流程：

settings --> SSH and GPG keys --> New SSH Key

打开 rsa_github.pub，将里面的内容复制到 Key 输入框中。

GitLab

GitLab 添加 SSH Key 流程：

Profile --> SSH keys

打开 rsa_gitlab.pub，将里面的内容复制到 Key 输入框中。
